Health Tracking System Using IoT

The proposed system consists of sensors and uses the internet to monitor and record patient's health data and medical information The proposed design basically provides a wearable sensor which records different health data and the data can be remotely accessed by the desired person.

The data collected by the sensors are first converted to digital signals using Arduino Uno and the digital data are displayed in the display board and are also sent to a website(a mobile application as project extension) using the ESP2866 Wi-Fi module.
Keywords:- Arduino Uno r3, Monitoring System, Sensors, Cloud, Wi-Fi module.
